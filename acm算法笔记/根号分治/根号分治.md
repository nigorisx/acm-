# 浅谈根号分治

[toc]

## 1.定义

根号分治是一种**根据数据规模分情况处理**的优化技巧，核心思想是**设置阈值B =$\sqrt{N}$，大情况暴力，小情况预处理**(打表)。

## 2.适用场景特征

- 有两个变量互相制约
- 暴力枚举其中一个会超时
- 可以按大小分类处理

## 3.题目示例和代码分析

```c++
#include <bits/stdc++.h>
using namespace std;
#define int long long
#define endl "\n"
const int B=450;
vector<int>pos[B+1];
void solve() {
    int n;
    cin >> n;
    vector<int> a(n+1);
    int ans=0;
    for(int i=1;i<=B;i++)//清空枚举表
    {
        pos[i].clear();
    }
    for(int i=1;i<=n;i++)//重新产生表
    {
        cin>>a[i];
        if(a[i]<=B)pos[a[i]].push_back(i);
    }
    for(int i=1;i<=n;i++)//情况一，a[i]为大值
    {
        if(a[i]>B)
        for(int k=1,j=i+a[i];j<=n;k++,j+=a[i])ans+=a[j]==k;
    }
    for (int j=1;j<=n;j++)//情况二，a[j]为大值
    {
        if(a[j]>B)
        for(int k=1,p=j-a[j];p>=1;k++,p-=a[j])ans+=(a[p]==k&&a[p]<=B);
    }
    for(int x=1;x<=B;x++)//情况三，无大值
    {
        if(!pos[x].size())continue;
        for(int y=1;y<=B;y++)
        {
            int  d=x*y;
            if(d>=n)continue;
            for(auto t:pos[x])ans+=(t+d<=n&&a[t+d]==y);
        }
    }
    cout<<ans<<endl;
}  
signed main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    int t;
    cin >> t;
    while (t--) {
        solve();
    }
}
```

复杂度分析如下

$$
\begin{align}
&if(a[i]>B)枚举次数k=n/a[i]<n/B=\sqrt{n}\\
&if(a[j]>B)对称化构造，相同枚举次数\\
&if(a[i]<B\quad and\quad a[j]<B)打表复杂度O(n）\\
\\
&综上所述复杂度为O（n\sqrt{n})
\end{align}
$$
