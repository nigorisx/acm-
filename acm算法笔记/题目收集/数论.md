## p1 [对和][https://ac.nowcoder.com/acm/contest/128672/D](简单数论，特简单，结论题，long long长记性题，不开卡90%)

> [!note]
>
> >***题目描述***
> >
> >小苯有一个长度为 $n$ 的数组 $a_1, a_2, \dots, a_n$。他定义数组的“和谐值”为：
> >
> >$H(a) = \sum_{i=1}^{n} \sum_{j=i+1}^{n} \left\lfloor \frac{a_i + a_j}{2} \right\rfloor$
> >
> >你的任务是求出"和谐值" $H(a)$。
>
> >***思路***
> >
> >这题实际相当简单，只要列几个找规律即可。我们先考虑没有取整号，我们将去算此时的$H_p(a)=\sum_{i=1}^{n} \sum_{j=i+1}^{n}  \frac{a_i + a_j}{2}$,应该是等于 $\sum_{i=1}^{n} (n-1)\times a_i$,这个值显然带偏差，那我们能算出偏差吗。首先分析偏差来源，发现当每有两对和为奇数的数对时，就会多1的偏差，因为本来应该舍去两个0.5，却累加出了一个1.所以我们只要算出奇偶相加的数对数就可。我们设偶数个数为$cnt_0$，奇数个数为$cnt_1$，那么奇偶数对就是cnt0*cnt1.所以可得$H(a) = H_p(a) - \frac{cnt_0 \times cnt_1}{2}$
>
> >code
> >
> >```c++
> >#include <bits/stdc++.h>
> >using namespace std;
> >#define int long long
> >void solve()
> >{
> >        int n;
> >        cin>>n;
> >        vector<int>v(n+1);
> >        int cnt0=0,cnt1=0;int sum=0;
> >        for(int i=1;i<=n;i++)
> >        {
> >             cin>>v[i];
> >             sum+=v[i];
> >             if(v[i]%2==1)cnt1++;
> >             if(v[i]%2==0)cnt0++;
> >        }
> >        int ans=(n-1)*sum;
> >        int num=cnt0*cnt1;
> >        cout<<ans/2-num/2<<"\n";
> >}
> >signed main()
> >{
> >        int t;
> >        cin>>t;
> >        while(t--)
> >        {
> >             solve();
> >        }
> >}
> >```
> >

## p2.[Divisive Battle][#https://codeforces.com/contest/2200/problem/E]（简单博弈论和完全幂筛，一道想到方法写不来的题）

> [!note]
>
> >***题目描述***
> >
> >E. Divisive Battle
> >
> >Alice and Bob are playing game on array a initially containing n positive integers, with Alice going first.
> >
> >On each player's turn, if a is non-decreasing∗, the game immediately ends. Otherwise, the player can choose an element x from the array and positive integers 1<y,z<x such that x=yz and replace x in the array with two elements y and z (in any order at the original location of x). If no such move is possible, the game ends.
> >
> >Once the game ends, if a is non-decreasing, then Bob wins. Otherwise, Alice wins. Determine who will win the game if both players play optimally.
>
> >概括一下题意，Alice想让这个数组非降，而Bob反之。显然Alice每次的最优策略是把z拆成尽可能小的x和尽可能大的y,这样可以让数组离Bob胜利的状态差距变大；Bob会去把大数拆成和小数相等，不然一定会有降序的部分。那么如果一个数如果不是完全幂，那么Bob的是必败的。除此之外，这个完全幂的底数在数组中也要升序，不然数组数与数之间的关系不合法。综上，我们要做的就是筛完全幂。这题可以用埃氏筛的变形，把原来的筛倍数改为筛幂次。复杂度O(nloglogn)
>
> > code
> >
> > ```c++
> > #include <bits/stdc++.h>
> > using namespace std;
> > #define int long long
> > const int N=1e6+5;
> > int f[N];int npri[N];
> > namespace SOL {
> >     using ll = long long;
> >     using pii = pair<int, int>;
> >     using vi = vector<int>;
> >     const int INF = 0x3f3f3f3f;
> >     const ll LLINF = 0x3f3f3f3f3f3f3f3fLL;
> > 
> >     template<typename T, typename L>
> >     bool chkmax(T &x, L y) { if (x < y) return x = y, true; return false; }
> >     template<typename T, typename L>
> >     bool chkmin(T &x, L y) { if (y < x) return x = y, true; return false; }
> >     
> >     void solve() {
> >         int n;
> >         cin >> n;
> >         vector<int> a(n+1);
> >         for (int i = 1; i <= n; ++i) {
> >             cin >> a[i];
> >         }
> >         bool flag=true;
> >         for(int i=1;i<=n-1;i++)
> >         {
> >             if(a[i]>a[i+1])flag=false;
> >         }
> >         if(flag)
> >         {
> >             cout<<"Bob"<<"\n";
> >             return;
> >         }
> >         for(int i=1;i<=n;i++)
> >         {
> >             if(f[a[i]]==-1)
> >             {
> >                 cout<<"Alice"<<"\n";
> >                 return;
> >             }
> >         }
> >         for(int i=1;i<=n-1;i++)
> >         {
> >             if(f[a[i]]>f[a[i+1]])
> >             {
> >                 cout<<"Alice"<<"\n";
> >                 return;
> >             }
> >         }
> >         cout<<"Bob"<<"\n";
> >     }
> > }
> > 
> > signed main() {
> >     ios::sync_with_stdio(false), cin.tie(0), cout.tie(0);
> >     int t;
> >     cin >> t;
> >     for(int i=1;i<=1000000;i++)f[i]=-1;
> >     f[1]=1;
> >     for(int i=2;i<=1000000;i++)
> > 	{
> > 		if(!npri[i])
> > 		{
> > 			for(int j=i*i;j<=1000000;j+=i)npri[j]=1;//筛掉倍数 复杂度O(nlog logn)
> > 			for(int j=i;j<=1000000;j*=i)f[j]=i;//处理 复杂度小于埃氏筛
> > 		}
> > 	}//完全幂筛
> >     while (t--) {
> >         SOL::solve();
> >     }
> >     return 0;
> > }
> > ```
