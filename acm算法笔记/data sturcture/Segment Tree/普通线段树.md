# 线段树

[toc]

## 定义

- 线段树是用来维护**区间信息**的数据结构
- 线段树可以在 $O(\log N)$ 的时间复杂度之内实现单点修改、区间修改、区间查询、区间求和、求最大值、最小值等操作



## 结构

- 线段树的本质是一颗二叉树,所以假如我们用$d_i$表示线段树节点i的值
- $d_i$ 的左儿子是 $d_{2\times i}$ ，右儿子是 $d_{2\times i +1}$ ，如果父节点表示的是区间 $\left[s,t\right]$ ，那么 左儿子节点表示的区间是 $\left[s,\frac{s+t}{2} \right]$ ，右儿子表示的区间就是 $\left[\frac{s+t}{2}+1,t\right]$ 
- 因此，我们考虑用递归建树



## 操作

### 建树

```c++
#define int long long
int d[MAXN*4],num[MAXN*4]  \\d为线段树，num为原数组

void push_up(int x)
{
    d[x]=d[2*x]+d[2*x+1];\\这里的运算符由不同的用途动态改变
}

void build(int x,int l,int r)
{
    if(l==r)
    {
        d[x]=num[l];
        return ;
    }
    build(2*x,l,(l+r)/2);
    build(2*x+1,l,(l+r)/2+1,r);
    push_up(x);
}
```

### 区间修改

- 如果要修改区间 $\left[l,r\right]$ ，把其中所有节点都遍历一遍的话时间复杂度太高，所以我们引入**懒惰标记**
- 简单来说，就是我们延迟对这个节点的修改，先把修改的信息打上标记，但是不更新其子节点，直到下一次访问到子节点时才去更新信息

```c++
int lazy(MAXN+1);
void push_down(int x,int l,int r)
{
    int mid=(l+r)/2;
    d[x*2]+=(mid-l+1)*lazy[x];
    d[x*2+1]+=(r-mid)*lazy[x];
    lazy[2*x]+=lazy[x];
    lazy[2*x+1]+=lazy[x];
    lazy[x]=0;
}

void add(int x,int l,int r,int ql,int qr,int num)
{
    if(ql<=l&&r<=qr)
    {
        lazy[x]+=num;
        d[x]+=(r-l+1)*num;
        return ;
    }
    push_down(x,l,r);
    int mid=(l+r)/2;
    if(ql<=mid)add(x*2,l,mid,ql,qr,num);
    if(qr>mid)add(x*2+1,mid+1,r,ql,qr,num);
    \\一定是双if 不要写成if else if结构
    push_up(x);
}
```

### 区间查询

```c++
int query(int x,int l,int r,int ql,int qr)\\区间和
{
    int(ql<=l&&r<=qr)
    {
        return d[x];
    }
    int ans=0;
    int mid=(l+r)/2;
    push_down(x,l,r);\\懒标记下传
    if(ql<=mid)ans+=query(x*2,l,mid,ql,qr);
    if(qr>mid)ans+=query(x*2+1,mid+1,r,ql,qr);
    return ans;
}
```

## 动态开点线段树（权值线段树）

- **权值线段树**是一种以序列的数值为下标，节点统计的是对应区间 $\left[l,r\right]$ 中，这个值域所有数出现的次数

- 每个**叶子节点**的值： 代表 这个值的出现次数。
- **非叶子节点**的值：代表了某一个值域内，所有值出现次数的和

- 如果按照普通的线段树构造方法，对于一个取值范围为 $w$ 的序列，我们需要 $O(w\log w)$ 的空间，这对于许多题来说都是不够用的，所以我们需要**动态开点**
- 这很简单易懂，普通的线段树的左右儿子关系是 $2\times x$ 与 $2\times x+1$ ，那么动态开点就用 $ls_x$ 与 $rs_x$ 来替代，只有在遍历到这个节点时才给它新建节点值

```c++
#include <bits/stdc++.h>
using namespace std;
#define int long long
#define mid ((l) + (r) >> 1)

const int MAXN = 1e7 + 8;
const int lim = 1e9;

int d[MAXN];
int ls[MAXN], rs[MAXN], node = 0;
int rt = 0;
int n;

void push_up(int x)
{
    d[x]=d[ls[x]]+d[rs[x]];
    return ;
}

void update(int &x, int l, int r, int col)
{
    if (x == 0)
        x = ++node;
    if (l == r)
    {
        d[x]++;
        return;
    }
    if (col <= mid)
        update(ls[x], l, mid, col);
    else
        update(rs[x], mid + 1, r, col);

    push_up(x);
}

int query(int x, int l, int r, int val)
{
    if (x == 0)
        return 0;
    if (l > val) //如果l都比val大，那么x代表区间都比val大
        return d[x];
    int res = 0;
    if (val <= mid)//如果val比mid小 也许还有mid左侧值比val大
        res += query(ls[x], l, mid, val);
    res += query(rs[x], mid + 1, r, val);

    return res;
}// 这个是大于val的数目
```
