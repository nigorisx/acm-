# 并查集

## 定义

***并查集是一种用于管理元素所属集合的数据结构，实现为一个森林，其中每棵树表示一个集合，树中的节点表示对应集合中的元素．***

并查集支持两种操作

***合并***（$\text{Unite}$）：合并两个元素所属集合（合并对应的树）．

***查找***（$\text{Find}$):查询某个元素所属集合（查询对应的树的根节点），这可以用于判断两个元素是否属于同一集合．

## 普通并查集具体实现（已带有按秩合并和路径压缩）

### 1.模板

```c++
#include <bits/stdc++.h>
using namespace std;
const int N = 2e5 + 5;
int fa[N], rnk[N];  // rnk表示树的高度（秩）
int n, m;
int op, x, y;
void initial()
{
    for(int i=1;i<=m;i++)
    {
        fa[i]=i;
        rnk[i]=1;
	}
}
int find(int x) {
    if (fa[x] == x) return x;  // 已经是根节点
    return fa[x] = find(fa[x]);  // 路径压缩
}
void Union(int x, int y) {
    x = find(x), y = find(y);  // 查询各自的根节点
    if (x == y) return;  // 如果已经在同一个集合，直接返回
    // 按高度（秩）合并：将矮树合并到高树下
    if (rnk[x] > rnk[y]) {
        swap(x, y);  // 确保rnk[x] <= rnk[y]
    }
    fa[x] = y;  // 将x的根节点指向y
    // 如果两棵树高度相同，合并后高度会增加1
    if (rnk[x] == rnk[y]) {
        rnk[y]++;
    }
}
int main() {
    cin >> n >> m;
    // 初始化
    for (int i = 1; i <= n; ++i) {
        fa[i] = i;      // 每个元素独立成集合
        rnk[i] = 0;     // 初始高度为0（单节点树高度为0）
    }
    for (int i = 1; i <= m; ++i) {
        cin >> op >> x >> y;
        if (op == 1) {  // 合并操作
            Union(x, y);
        } else {        // 查询操作
            x = find(x), y = find(y);
            if (x == y) {
                cout << "Y" << endl;
            } else {
                cout << "N" << endl;
            }
        }
    }
    
    return 0;
}
```

### 2.复杂度分析

***初始化：复杂度接近 $\text{O}(n)$***

***Find函数：复杂度在次数多时接近 $\text{O}(\alpha(n))$,其中$\alpha$是反阿克曼函数，复杂度极低，基本认为是常数阶***

***Union函数：时间复杂度同样为 $\text{O}(\alpha(n))$***

$\therefore$总时间复杂度接近 $\text{O}(n+m\cdot \alpha(n))$,及其高效

## 生成树问题（Kruskal算法）：合并森林

### 对比Prime和kruskal

普利姆（Prime）算法是以 **点** 为基础，挑选与 **已有点** 相连的最小边。

克鲁斯卡尔（Kruskal）算法是以 **边** 为基础，先将边从小到大排列，从小到大的添加不构成「环路」的边。

### 代码执行过程

**算法执行过程中形成的各个连通分量，都确实是“自己那个连通分量的最小生成树**

```c++
#include <bits/stdc++.h>
using namespace std;
#define int long long 
const int  N=1e4+5;
int fa[N];int rnk[N];
int n,m,k;
struct node
{
    int u,v,w;
    bool operator<(const node& a)const
    {
        return w<a.w;
    }
};
void initial()
{
    for(int i=1;i<=n;i++)
    {
        fa[i]=i;
    }
}
int find(int x)
{
    if (x == fa[x])
	{
		return x;
	}
	return fa[x] = find(fa[x]);
}
void Union(int x,int y)
{
    x=find(x);y=find(y);
    if(fa[x]==fa[y])return ;
    if(rnk[x]>rnk[y])swap(x,y);
    fa[x]=y;
    if(rnk[x]==rnk[y])rnk[y]++;
}
node g[N];
signed main()
{
    int sum=0;
    cin>>n>>m>>k;
    if(k>n)
    {
        cout<<"No Answer";
        return 0;
    }
    if(n==k)
    {
        cout<<0;
        return 0;
    }
    int cnt=n;
    initial();
    for(int i=1;i<=m;i++)
    {
        int from,to,w;
        cin>>from>>to>>w;
        g[i]={from,to,w};
    }
    sort(g+1,g+m+1);
    for(int i=1;i<=m;i++)
    {
        int x=g[i].u;int y=g[i].v;
        if(find(x)==find(y))continue;
        else{
            cnt--;
            Union(x,y);
            sum+=g[i].w;
            if(cnt==k)
            {
                cout<<sum;
                return 0;
            }
        }
    }
    cout<<"No Answer";
}
```

## 带删除并查集（有自己写的插入函数）

**普通的并查集无法支持删除操作，是因为删除一个节点的时候，不可避免地会将以它为根的子树上所有节点都删除．为了解决这一问题，在带删除操作的并查集中，可以通过建立虚点的方法保证所有实际存储数据的节点总是叶子节点．为此，需要在初始化时，就为每个数据节点都建立一个虚点，并将数据节点的父节点设置为该虚点．由于每次合并两个集合时，都只会将两个集合的树根连接，所以，从始至终只有虚点会有子节点．这就保证了删除一个节点时，不会误删其他节点．**

***注意，删除单个节点后，需要重新为该节点建立一个虚点作为其父节点；否则，无法正确执行后续的合并和删除操作．***

```c++
#include <iostream>
using namespace std;
#define int long long 
int n,m;
const int N=1e6+5;
int fa[N*4];int rnk[N*4];int sum[N*4];
int d1;
void init()
{
    d1 = 2 * n+1;
    for(int i = 1; i <= n; i++)
    {
        fa[i] = i + n;    
        fa[i + n] = i + n; 
        rnk[i + n] = 1;    
        sum[i + n] = i;   
    }
}//初始化，把所有根节点设置为虚节点
int find(int x)
{
    if(fa[x]==x)return x;
    else return fa[x]=find(fa[x]);
}//带路径压缩的查找
void merge(int x,int y)
{
    x=find(x);y=find(y);
    if(x==y)return;
    if(rnk[x]>rnk[y])swap(x,y);
    fa[x]=y;
    rnk[y]+=rnk[x];
    sum[y]+=sum[x];
}//合并操作
void erase(int x)
{
    int y=find(x);
    rnk[y]--;
    sum[y]-=x;
    fa[x]=d1;
    rnk[d1]=1;
    fa[d1]=d1;
    sum[d1]=x;
    d1++;
}//将当前元素挪到虚节点右侧的区域
void ist(int a,int b)
{
    if(find(a)==find(b))return ;
    erase(a);
    merge(d1-1,b);//由于此时a在右侧d1-1处，所以合并时要修改位置
}//用erase和merge操作合成insert操作
signed main()
{
    while(scanf("%lld%lld",&n,&m)!=EOF)//uva特性
    {   
        init();//初始化，不要忘！不要忘！不要忘！
        for(int i=1;i<=m;i++)
        {
            int opt,a,b;
            cin>>opt;
            if(opt==1)//合并
            {
                cin>>a>>b;
                merge(a,b);
            }
            else if(opt==2)//插入
            {
                cin>>a>>b;
                ist(a,b);
            }
            else{
                cin>>a;
                cout<<rnk[find(a)]<<" "<<sum[find(a)]<<endl;//输出节点数和总和
            }
        }
    }
}
```

## 带权并查集

**如果两个节点之间的边具有权值，那么在路径压缩和合并时权值需要进行修改，一般用子节点来储存它与其父节点之间边的权值。在路径压缩时将根与子节点之间路径权值之和记录到子节点上，在合并两个节点所在集合时，需要计算两个根节点之间新连接的边的权值。权的含义是节点间的关系,可以是差分关系，距离关系，逻辑关系etc,权值定义不同，维护的方程也不同，但维护的都是向量方程**

```c++
#include<bits/stdc++.h>
using namespace std;
int const N=1e4+7;
int n,m;
int f[N],w[N],dis[N];  //w表示集合根节点的权值  //dis[i]表示i与父节点（权值）的差值，有点类似差分
int find(int x){
    if(f[x]!=x){
        int fx=f[x];
        f[x]=find(f[x]);
        dis[x]+=dis[fx];   //路压维护权值 //以后难一点的题目，这一步可以画图，举例子，来推出如何维护
    }
    return f[x];
}
void merge(int a,int b){
    int fx=find(a),fy=find(b);
    f[fx]=fy;
    dis[fx]=w[fx]-w[fy]; //合并维护权值 //以后难一点的题目，这一步可以画图，举例子，来推出如何维护
    //主要是维护数学关系w[i]=dis[i]+w[fa[i]] 不变
    //每一题关系都不同,不能死套板子；
    //有时关系维护靠的是子节点，那么就要添加子节点的关系变量d
}
int main(){
    cin >> n >> m;
    for(int i=1;i<=n;++i) f[i]=i;
    int op,a,b;
    while(m--){
        cin >> op >> a >> b;
        if(op==1){
            if( find(a)!=find(b) ) merge(a,b);
        }
        else w[find(a)]+=b;
    }
    for(int i=1;i<=n;++i){
        cout << w[find(i)]+dis[i] << " ";
    }
    return 0;
}

```

## 拓展域并查集

**拓展域并查集**是一种数据结构，它在传统并查集的基础上进行了扩展，以处理更复杂的关系问题。在标准的并查集中，我们通常关注的是元素是否属于同一个集合。而在拓展域并查集中，我们不仅要处理元素是否属于同一个集合的问题，还要处理元素之间可能存在的多种关系，例如相互排斥的关系。常见例题，[食物链问题](#https://www.luogu.com.cn/problem/P2024) ,[团伙问题](#https://www.luogu.com.cn/problem/P1892),以食物链问题为例。

### P2024 [NOI2001] 食物链

 **题目描述**

动物王国中有三类动物 $A,B,C$，这三类动物的食物链构成了有趣的环形。$A$ 吃 $B$，$B$ 吃 $C$，$C$ 吃 $A$。

现有 $N$ 个动物，以 $1 \sim N$ 编号。每个动物都是 $A,B,C$ 中的一种，但是我们并不知道它到底是哪一种。

有人用两种说法对这 $N$ 个动物所构成的食物链关系进行描述：

- 第一种说法是 `1 X Y`，表示 $X$ 和 $Y$ 是同类。
- 第二种说法是 `2 X Y`，表示 $X$ 吃 $Y$。

此人对 $N$ 个动物，用上述两种说法，一句接一句地说出 $K$ 句话，这 $K$ 句话有的是真的，有的是假的。当一句话满足下列三条之一时，这句话就是假话，否则就是真话。

- 当前的话与前面的某些真的话冲突，就是假话；
- 当前的话中 $X$ 或 $Y$ 比 $N$ 大，就是假话；
- 当前的话表示 $X$ 吃 $X$，就是假话。

你的任务是根据给定的 $N$ 和 $K$ 句话，输出假话的总数。

 输入格式

第一行两个整数，$N,K$，表示有 $N$ 个动物，$K$ 句话。

第二行开始每行一句话。格式见题目描述与样例。

 输出格式

一行，一个整数，表示假话的总数。

 输入输出样例 #1

 输入 #1

```
100 7
1 101 1
2 1 2
2 2 3
2 3 3
1 1 3
2 3 1
1 5 5

```

 输出 #1

```
3

```

```c++
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
struct DSU
{
    vector<int> p, siz;
    DSU(int n) : p(n + 1), siz(n + 1, 1)
    {
        iota(p.begin(), p.end(), 0);
    }
    int find(int x)
    {
        while (x != p[x])
            x = p[x] = p[p[x]];
        return x;
    }
    bool merge(int x, int y)
    {
        int fx = find(x);
        int fy = find(y);
        if (fx == fy)
            return false;
        siz[fx] += siz[fy];
        p[fy] = fx;
        return true;
    }
};
void eachT()
{
    int n, m1, m2;
    cin >> n >> m1 >> m2;
    DSU dsu1(n + 1);
    DSU dsu2(n + 1);
    vector<int> u(m1 + 1);
    vector<int> v(m1 + 1);
    for (int i = 1; i <= m1; i++)
        cin >> u[i] >> v[i];
    int ans = 0;
    for (int i = 1; i <= m2; i++)
    {
        int u, v;
        cin >> u >> v;
    }
    for (int i = 1; i <= m1; i++)
    {
        if (dsu2.find(u[i]) != dsu2.find(v[i]))
            ans++;
        else
            dsu1.merge(u[i], v[i]);
    }
    int res1 = 0, res2 = 0;
    for (int i = 1; i <= n; i++)
    {
        if (dsu1.find(i) == i)
            res1++;
        if (dsu2.find(i) == i)
            res2++;
    }
    ans += res1 - res2;
    cout << ans << '\n';
}
int main()
{
    ios::sync_with_stdio(false);
    cin.tie(0), cout.tie(0);
    int t;
    cin >> t;
    while (t--)
    {
        eachT();
    }
    return 0;
}
```
