## 矩阵快速幂

```c++
struct Matrix {
    int r, c;
    ll mt[N][N];
    Matrix() {
        memset(mt, 0, sizeof(mt));
    }
    Matrix operator*(const Matrix& other) const {
        Matrix res;
        res.r = r;                   
        res.c = other.c;            
        for(int i = 0; i < r; i++) {
            for(int j = 0; j < other.c; j++) {
                for(int p = 0; p < c; p++) {
                    res.mt[i][j] = (res.mt[i][j] + 
                                   mt[i][p] * other.mt[p][j]) % M;
                }
            }
        }
        return res;
    }
};
//0-based
Matrix qpow(Matrix a, ll b) {
    Matrix ans;
    ans.r = 2; 
    ans.c = 2;
    ans.mt[0][0] = 1;
    ans.mt[1][1] = 1;
    while(b) {
        if(b % 2) ans = ans * a;
        a = a * a;
        b /= 2;
    }
    return ans;
}
```

## 拓展欧几里得定理

用于解$ax+by=gcd(a,b)$的一组特解（**都要去 mod$\frac{b}{gcd(a,b)}$,不要mod b**)

$\frac{b}{gcd(a,b)}$时x变化的最小正变化周期，同理$\frac{a}{gcd(a,b)}$时y变化的最小正变化周期（可证）

模板

```c++
int exgcd(int a, int b, int &x, int &y) {
    if (b == 0) { x = 1, y = 0; return a; }
    int d = exgcd(b, a % b, y, x);
    y -= (a / b) * x;
    return d;//d是gcd（a,b)
}
```

***从上方可以粗略推出当$gcd(a,b)|c$时，$ax+by=c$有解，即*裴蜀定理**

所以可以从中推得求解$ax+by=c$的方法（一般会将c置为正数）（求最小正整数解的方法）

1.求解$ax+by=gcd(a,b)$，得到一组特解$x_{0},y_{0}$

2.左右同乘上$\frac{c}{gcd(a,b)}$,得到$a*x_{0}*\frac{c}{gcd(a,b)}+b*y_{0}*\frac{y_{0}}{gcd(a,b)}=c$

3.解得特解为$x^{\prime}=x_{0}*\frac{c}{gcd(a,b)} \quad y^{\prime}=y_{0}*\frac{y_{0}}{gcd(a,b)}$

4.解得通解为$x=x^{\prime}+p*\frac{b}{d}\quad y=y^{\prime}-p*\frac{a}{d}\quad(p\in z)$

5 $\because \frac{b}{d}不一定为正数$，$\therefore$取模时取其绝对值$\abs{\frac{b}{d}}$

6.因此最小正整数解为$x_{min}=(x\mod{t}+\quad t)\mod{t}\quad(t=\abs{\frac{b}{d}})$

```c++
#include <bits/stdc++.h>
#define int long long
#define endl "\n"
using namespace std;
int exgcd(int a, int b, int &x, int &y) {
    if (b == 0) { x = 1, y = 0; return a; }
    int d = exgcd(b, a % b, y, x);
    y -= (a / b) * x;
    return d;
}
signed main()
{
   int x,y,m,n,l;
   cin>>x>>y>>m>>n>>l;
   int a=(m-n);int b=l;
   int c=y-x;
   if(c<0)
   {
    a=-1*a;
    b=-1*b;
    c=-c;
   }
   int x0,y0;
   int d=exgcd(a,b,x0,y0);
   if(c%d!=0)
   {
    cout<<"Impossible";
    return 0;
   }
   int x1=x0*c/d;
   int y1=y0*c/d;
   int mod=abs(b/d);
   int ans=((x1)%mod+mod)%mod;
   cout<<ans;
}
```

void solve()
{
    int a,b,c;
    cin>>a>>b>>c;
    int x,y;
    int d=exgcd(a,b,x,y);
    if(c%d!=0)
    {
        cout<<"-1"<<endl;
        return ;
    }
    int x1=x*(c/d);
    int y1=y*(c/d);
    int tx=b/d;int ty=a/d;
    int n=ceil(1.0*(1-x1)/tx);
    int m=floor(1.0*(y1-1)/ty);
    if(n>m)
    {
        cout<<x1+n*tx<<" "<<y1-m*ty<<endl;
        return ;
    }
    else
    {
        int num=m-n+1;
        int xmin=x1+n*tx;
        int ymax=y1-n*ty;
        int ymin=y1-m*ty;
        int xmax=x1+m*tx;
        cout<<num<<" "<<xmin<<" "<<ymin<<" "<<xmax<<" "<<ymax<<endl;
        return;
    }
}

## 同余与逆元

### 同余

若 m|(a-b),记作a$\equiv$b(mod m)

- 消去律：若：若  $ac≡bc (mod\,m)$，则  $a≡b(mod\quad \frac{m}{gcd(c,m)} )$。只有当$gcd(c,m) = 1$ 时⽅可直接消去。

### 逆元

若$ax≡1 (mod\,m)$，则称x为a在模m下的***逆元***。

- exgcd 求 ax+my=1的特解，最好将 x调到[0,m)范围内称为乘法标准逆元

- 费马小定理$a^{p-2}(mod\,p)(p为质数)$

### 线性逆元

快速推出1~n范围中所有数在模P下的逆元（P为质数）

递推：

$$
inv[i]=(P-\lfloor P/i \rfloor)\cdot inv[P  \quad (mod\quad i)]\quad (mod\quad P)
$$

$O(n)$复杂度的预处理，效率高

## 数论三大定理

### 1.费马小定理(Fermat's Little Theorem)

若$p$为质数，且$\gcd(a,p)$=1(即a,p互质）,则$a^{p-1}\equiv1(mod\quad p)$

```c++
int pow(int a, int b, int m) {
  long long res = 1, po = a;
  for (; b; b >>= 1) {
    if (b & 1) res = res * po % m;
    po = po * po % m;
  }
  return res;
}//快速幂加速
int inv(int a, int p) { return pow(a, p - 2); }
```



### 2.欧拉定理(Euler's Theorem)

若$\gcd(a,m)=1$,则$a^{\phi{(m)}}\equiv1(mod\quad m)$

### 3.威尔逊定理

对于整数$p,(p−1)!≡−1 (mod\quad p)$是$p$为质数的充分必要条件。

## 筛法，唯一分解定理，概率和组合

### 筛法

#### 1.埃氏筛

```c++
vector<int> prime;
bool is_prime[N];

void Eratosthenes(int n) {
  is_prime[0] = is_prime[1] = false;//0,1不是素数
  for (int i = 2; i <= n; ++i) is_prime[i] = true;//一开始都假设是素数
  for (int i = 2; i <= n; ++i) {
    if (is_prime[i]) {//如果当前数是素数
      prime.push_back(i);
      if ((long long)i * i > n) continue;
      for (int j = i * i; j <= n; j += i)//筛时保证要筛的数大于当前，否则会重复筛
        // 因为从 2 到 i - 1 的倍数我们之前筛过了，这里直接从 i
        // 的倍数开始，提高了运行速度
        is_prime[j] = false;  // 是 i 的倍数的均不是素数
    }
  }
}
```

复杂度$O(n\log{\log{n}})$

#### 2.线性筛（欧拉筛）

复杂度 $O(n)$

```c++
vector<int> pri;
bool not_prime[N];

void pre(int n) {
  for (int i = 2; i <= n; ++i) {
    if (!not_prime[i]) {
      pri.push_back(i);
    }
    for (int pri_j : pri) {
      if (i * pri_j > n) break;
      not_prime[i * pri_j] = true;
      if (i % pri_j == 0) {
        // i % pri_j == 0
        // 换言之，i 之前被 pri_j 筛过了
        // 由于 pri 里面质数是从小到大的，所以 i 乘上其他的质数的结果一定会被
        // pri_j 的倍数筛掉，就不需要在这里先筛一次，所以这里直接 break
        // 掉就好了
        break;
      }
    }
  }
}

```

### 唯一分解定理

***任何大于1的自然数都可以唯一地表示为素数的乘积***

$n=p^{c_{1}}p^{c_{2}}p^{c_{3}}p^{c{4}}\dots(p为质数)$

所有因子之和就是将每个p求等比数列求和然后乘起来
