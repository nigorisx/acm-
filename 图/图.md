## 存储结构

- **临接数组**:一种以$n^{2}$大小的数组存储节点间链接关系的数据结构



- **临接表(`adj`)**:一种记录相连节点链接关系的动态数组,一般依赖于动态数组$vector$来实现,会将$vector$开成二维的,第一维度固定为节点数目,也可理解为此时的起点.[^02]

## 同源最短路

$\text{Dijkstra}$ 算法

算法内容:$Dijkstra$算法将一张图内所有结点分为了两个集合，其中一个集合内放已经确定了起点到其的最短路的点，另一个放还没有确定最短路的点，我们分别记为$S$和$T$。为了记录源点到每个点的最短路，我们要用一个数组 。刚开始的时候所有的点都属于集合$T$ ，接下来，每一次操作都从$T$中取出最短路长度最小的点放入$S$集合中，对其进行**松弛**操作，什么是松弛呢？我们进行如下定义：对于一条边，设其边权为$W$,则 $dis(v)=min(dis(v),dis(u)+w)$。这个操作就叫做松弛，而它的含义是显而易见的，就是当我们走到一个点的时候考虑要不要去走某一条边。

***代码模板***（*邻接表版本*）

```c++
#include <bits/stdc++.h>
using namespace std;
#define int long long
const int  N = 2e5 + 7, INF =0x3f3f33f3f ;
struct edge { int v, w; };
vector<edge> e[N];
priority_queue<pair<int,int>, vector<pair<int,int> >, greater<pair<int,int> > > q;
int  n, m, s;
int  dis[N];
bool used[N];
void Dijkstra(int s) {
    for (int i = 1; i <= n; ++i) { dis[i] = INF; }
    dis[s] = 0;
    q.push({0, s});
    while (!q.empty()) {
        auto t = q.top(); q.pop();// 按照dis排序，每次取出堆顶的点
        int u = t.second;
        if (used[u]) { continue; }// 如果在S集合中就扔掉不管
        used[u] = true;// u完成使命放入S集合中
        for (auto it : e[u]) {
            int  v = it.v, w = it.w;
            if (dis[v] > dis[u] + w) {
                dis[v] = dis[u] + w;
                q.push({dis[v], v});
            }
        }
    }
    return;
}

signed main()
{
    cin>>n>>m>>s;
    for(int i=1;i<=m;i++)
    {
        int  from,to,w;
        cin>>from>>to>>w;
        e[from].push_back({to,w});
    }
    Dijkstra(s);
    for(int i=1;i<=n;i++)
    {
        cout<<dis[i]<<" ";  
    }
}
```

代码示例（链式前向星）

```c++
#include <bits/stdc++.h>
using namespace std;
#define int long long
const int N = 2e5 + 7, INF = 0x3f3f3f3f3f3f3f3f; // 修正INF，使用8字节的极大值
priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> q;
int n, m, s;
int dis[N];
bool used[N];
struct Edge {
    int to, w, next;
} edge[N]; // 注意：N要足够大，通常边数是点数的几倍
int head[N], cnt;
// 添加有向边
void add_edge(int u, int v, int w) {
    edge[cnt].to = v;      // 终点
    edge[cnt].w = w;       // 边权
    edge[cnt].next = head[u]; // 指向u的下一条边
    head[u] = cnt++;       // 更新u的邻接表头
}
void Dijkstra(int s) {
    for (int i = 1; i <= n; ++i) {
        dis[i] = INF;
        used[i] = false;   // 初始化
    }
    dis[s] = 0;
    q.push({0, s});
    
    while (!q.empty()) {
        auto t = q.top();
        q.pop();
        int u = t.second;
        
        if (used[u]) continue;
        used[u] = true;
        
        // 遍历u的所有邻接边 - 链式前向星的遍历方式
        for (int i = head[u]; i != -1; i = edge[i].next) {
            int v = edge[i].to;
            int w = edge[i].w;
            
            if (dis[v] > dis[u] + w) {
                dis[v] = dis[u] + w;
                q.push({dis[v], v});
            }
        }
    }
}
signed main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cin >> n >> m >> s;
    // 初始化head数组为-1
    memset(head, -1, sizeof(head));
    cnt = 0; // 初始化边计数器
    for (int i = 1; i <= m; i++) {
        int from, to, w;
        cin >> from >> to >> w;
        add_edge(from, to, w);
        // 如果是无向图，需要再加一条反向边：
        // add_edge(to, from, w);
    }
    Dijkstra(s);
    for (int i = 1; i <= n; i++) {
        if (dis[i] >= INF) 
            cout << "2147483647 "; // 题目通常要求输出这个值
        else 
            cout << dis[i] << " ";
    }
    cout << endl;

    return 0;
}
```

## 拓扑排序

在图论中,***拓扑排序***(Topological Sorting)是一个***有向无环图***(DAG, Directed Acyclic Graph)的所有顶点的线性序列。且该序列必须满足下面两个条件：

- 1.每个顶点出现且只出现一次。

- 2.若存在一条从顶点 A 到顶点 B 的路径，那么在序列中顶点 A 出现在顶点 B 的前面。

***拓扑排序的目标是将所有节点排序，使得排在前面的节点不能依赖于排在后面的节点．***

***同时，拓扑排序还可以判断图中是不是有环*** (**当所有节点的入度都更新完后,即度为0的队列已空,但入度的数组中仍有非零值，则图中有环**

代码思想:统计每个节点的***入度*** [^03],同时用临接表储存整幅图的结构,$\text{dfs}$遍历图,删去入度为0的节点,并更新相关点的入度.
code模板

```c++
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
#include <queue>
using namespace std;
vector<int> v;//入读数组
vector<vector <int> > v1;//邻接表
int main()
{
    int n;
    cin >> n;
    v.resize(n+1);
    v1.resize(n+1);
    queue<int>q;
    for(int i=1;i<=n;i++)
    {
        int x;cin>>x;
        while(x!=0)
        {
            v1[i].push_back(x);
            v[x]++;
            cin>>x;
        }
    }
    vector<int>order;
    for(int i=1;i<=n;i++)
    {
        if(v[i]==0)
        {
            q.push(i);
            order.push_back(i);
        }
    }
    while(!q.empty())
    {
        int p=q.front();
        q.pop();
        for(auto u:v1[p])
        {
            v[u]--;
            if(v[u]==0)
            {
                order.push_back(u);
                q.push(u);
            }
        }
    }
    for(int i=0;i<order.size();i++)
    {
        cout<<order[i]<<" ";
    }
}

```
