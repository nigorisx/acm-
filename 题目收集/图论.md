## p1.***Floyd传递闭包***（bitset优化）

> [!note]
>
> 前置知识：传递闭包
>
> 从数学上来说，传递闭包是在集合x上求包含关系r的最小传递关系。从关系图的角度来说，就是如果原关系图上有i到j的**路径**，则其传递闭包的关系图上就应有从i到j的**边**。***通俗地讲，就是确定每个点是否能到达其他每个点。***
>
> Floyd算法实现，E是关系矩阵
>
> ```
> for (int k = 1; k <= n; ++k)
> for (int i = 1; i <= n; ++i)
> for (int j = 1; j <= n; ++j)
> if (E[i][k] && E[k][j])
> E[i][j] = 1;
> ```

>[!note]
>
>bitset优化连通数判断
>
>思路：在关系图中，我们已知直接相连的关系，假如i可以到k，那么k能到的节点，i一定可以间接相连，用位运算表示，就是f[i]|=f[i]|f[k]
>
>因为本身Floyd复杂度为O(n^3),而bitset可以优化大概1/8，所以在极限复杂度下可以优化到1e9范围内。
>
>```c++
>#include <bits/stdc++.h>
>using namespace std;
>#define int long long
>namespace SOL
>{
>    using ll = long long;
>    using pii = pair<int, int>;
>    using vi = vector<int>;
>    using vvi = vector<vector<int>>;
>    const int INF = 0x3f3f3f3f;
>    const ll LLINF = 0x3f3f3f3f3f3f3f3fLL;
>    template<typename T, typename L>
>    bool chkmax(T &x, L y) { if (x < y) return x = y, true; return false; }
>    template<typename T, typename L>
>    bool chkmin(T &x, L y) { if (y < x) return x = y, true; return false; }
>    void solve()
>    {
>        int n;
>        cin>>n;
>        vector<bitset<2005>>f(n+1);
>        int ans=0;
>        for(int i=1;i<=n;i++)
>        {
>            for(int j=1;j<=n;j++)
>            {
>                char x;
>                cin>>x;
>                if(x=='1')f[j].set(i);
>            }
>            f[i].set(i);
>        }
>        for(int k=1;k<=n;k++)
>        {
>            for(int i=1;i<=n;i++)
>            {
>                if(f[i][k])f[i]|=f[k];
>            }
>        }
>        for(int i=1;i<=n;i++)
>        {
>            ans+=f[i].count();
>        }
>        cout<<ans;
>    }
>}
>signed main()
>{
>    ios::sync_with_stdio(false), cin.tie(0), cout.tie(0);
>    int t;
>    t=1;
>    while(t--)
>    {
>        SOL::solve();
>    }
>}
>```
